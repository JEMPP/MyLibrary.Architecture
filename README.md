# MyLibrary Architecture Standards
`MyLibrary.Architecture`

Dieses Repository definiert die Architekturstandards
für alle MyLibrary-Projekte.

Ziele:

- Einheitliche Projektstruktur
- Wiederverwendbare Bibliotheken
- Hohe Testbarkeit
- GitHub-Ready
- NuGet-Ready
- SOLID-Prinzipien
- Security by Default
